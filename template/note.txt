템플릿
    웹 애플리케이션에서 작은 뷰에 해당한다. 컴포넌트 마다 템플릿이 있고 컴포너트들이 조합되어 하나의 큰 뷰가 만들어 진다.

컴포넌트 추가 필요
    import { Component } from '@angular/core';

컴포넌트 장식자와 컴포넌트 클래스를 추가한다.
    @Component({
        selector
        template
    })

표현식
    바인딩 기호 {{}} 내에서 사용된다.

명령식
    (click) = onSave();

연산자 사용불가 목록
    =, +=, -=
    ++, --
    new
    비트 연산자

파이프 연산자 : 출력 형태를 지정할 때 사용
    {{ name | uppercase}}

안전 탐색 연산자 : ?.
    {{my?.a}}
